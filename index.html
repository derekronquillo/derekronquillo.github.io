About Me

I'm Derek. Software engineer and Artist, born in the Philippines.

I started coding at 10. Now I build software that runs fast, scales clean, and actually solves problems. No fluff. No fragile hacks. Just solid, maintainable code that works where it matters. I speak English, Korean, and Tagalog, so yes! I code multilingual too.

What’s the point of a pretty UI if users can’t even use it? I care about clean logic, performance, and accessibility—because real engineering is more than visuals.

When I’m not coding, I play instruments, study new tech, or chill with Netflix.

Coffee or hot chocolate? Nah. Flavored soy milk. Acidic stomach. Deal with it. I don’t talk big—I just build. I create systems that work. Fast. Reliable. Done.
